Dataset Requirements

Created on: 05.10.2022, 13:05

  • Data are in the smart cities domain
    • They belong to the Comunidad the Madrid and are about Smart Medical Treatment.
  • Data are available as a CSV file
  • Data have an open license so it can be published
    • Their license is [https://creativecommons.org/licenses/by/4.0/legalcode](Creative Commons).
  • Data can be easily linked with generic real-world entities (e.g., locations)
    • The data contains facility addresses.
  • Documentation exists for the data (optional)
    • Each dataset has a dedicated web page with additional information and metadata
  • Data can come from multiple data sources (optional)
    • They both are from https://www.comunidad.madrid/gobierno/datos-abiertos
